# my-website

个人网页项目，使用纯 HTML/CSS/JS 构建，部署于 Vercel。

## 🚀 本地运行

用任意浏览器打开 `index.html` 即可预览。

## 📦 部署

推送到 GitHub 后自动部署到 Vercel。

## 📁 项目结构

```
my-website/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
├── assets/
├── README.md
├── .gitignore
└── .github/
    └── PULL_REQUEST_TEMPLATE.md
```
